Bolt Action at Leeds Night Owls

Bolt Action is a World War 2 skirmish game made by Warlord Games using 28mm miniatures that is very popular at Night Owls. With the recent release of third edition, Night Owls have begun to move over to the newest edition of Bolt Action.

Bolt Action Games at Leeds Night Owls

Photos of Bolt Action games taking place at Leeds Night Owls.

Bolt Action being played at Night Owls. This is a game of third edition, a blue on blue German game.

Bolt Action tournament hosted by Night Owls back in 2018.

Last game of Bolt Action version 2 before the release of version 3.
